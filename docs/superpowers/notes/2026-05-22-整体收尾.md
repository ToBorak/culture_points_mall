## 项目亮点

1. **价值观引擎为骨架**：每一份积分有归属、每一个徽章有代表、每一次活动有指向。6 维度 × 4 阶徽章自动触发。
2. **同一份业务工具，两种协议出口**：HR-Agent (LLM Function Calling) + MCP Server (JSON-RPC over SSE) 共享同一份 Tool Registry。9 个工具一次注册，两个协议自动可见。
3. **TCC + Redis 双保险**：「未中奖不扣分」严格执行。集成测试连续抽 10 次，win/miss 计数与积分扣减完美一致。
4. **动漫风设计系统**：6 primitives（Panel/Stamp/Shout/KW/Halftone/SpeedLines）+ 5 components（ComicButton/DimChip/RadarChart3D/2D/BadgeWall/BlindboxWheel）+ Storybook 全覆盖。
5. **钉钉适配层 Mock/Real 可切换**：演示阶段走 Mock（含 admin 推送可视化面板），生产切 corpId/appKey/appSecret 即可。
