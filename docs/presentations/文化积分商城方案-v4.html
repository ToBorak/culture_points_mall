配额

应对：路演时坦诚说明 — 工程严谨度反而是加分项，已计入预算。

🔴 MCP 客户端兼容
环境

应对：Demo 当天用 Claude Desktop 已知稳定版本，预录视频作为兜底。

🔴 自研签到防作弊
演示

应对：演示用 GPS + 二维码 + 随机题足够，路演不深究细节。

🔴 跨企业数据冷启动

应对：内置 3 家虚拟对手企业，演示足够 — Phase 1 不需要真实合作伙伴。

外部

机会

应对：自研轻量 JSON-RPC 实现 — 反而成为技术亮点，转危为机。
